> Object oriented python wrapper for adb protocol.

## Install
```
$ pip install simpleadb
```

## Usage

```Python
>>> import simpleadb
>>> adb_server = simpleadb.AdbServer()
>>> devices = adb_server.devices()
>>> emulator = devices[0]
>>> emulator
'emulator-5554'
>>> emulator.root()
restarting adbd as root
0
>>> emulator.reboot()
```
